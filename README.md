# 🌿 Paradise Nursery - Shopping Application

A dynamic e-commerce web application for an online plant shop called Paradise Nursery. Users can browse plants, view details, add items to a shopping cart, and manage cart quantities.

## Features
- Browse plants by categories (Air Purifying, Low Maintenance, Flowering Plants)
- View plant details with images, names, descriptions, and prices
- Add plants to shopping cart with quantity management
- Dynamic cart total calculation
- Responsive navigation bar

## Technologies Used
- React.js
- Redux Toolkit
- CSS3
